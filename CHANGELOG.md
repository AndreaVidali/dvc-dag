# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and uses the emoji approach of [iStat Menus version history](https://bjango.com/mac/istatmenus/versionhistory/) for better readability.

Here is the syntax:

> - ❇️ (added) for new features.
> - ⚙️ (improved/changed) for improvements and changes in existing functionalities.
> - ⚡️ (fixed) for any fixes.
> - ❌ (removed) for now removed features.
